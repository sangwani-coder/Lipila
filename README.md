# SKOOLPAY
School Fee Collection System
## Requirements:
- Python3
- Flask

## Usage Linux:
**Set ENV Variables**
- $ cd SKOOLPAY
- $ export FLASK_ENV=development
- $ export FLASK_APP=app
- $ flask run

## Usage Windows:
- cd SKOOLPAY
- $ python app.py

## Language
- Python3
## Framework
- Flask
## Database
Sqlite in developmeent and MySQL in production

## Features
- Collect Fees
- Add/Remove students
- Update details/Admin
- Adds payments to DB automatically
- Print reports
